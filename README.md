# まるもちシマエナガ

かわいい、まん丸のシマエナガをモチーフにしたCodex用の16-bit風ドット絵ペットです。

## 使い方

1. このリポジトリをダウンロードまたはcloneします。
2. `marumochi-shimaenaga-pixel` フォルダを、Codexのカスタムペット用フォルダへコピーします。

   ```text
   %USERPROFILE%\.codex\pets\marumochi-shimaenaga-pixel\
   ```

3. `pet.json` と `spritesheet.webp` が同じフォルダにあることを確認します。
4. Codexを再起動し、設定 → パーソナライズ → ペットから「まるもちシマエナガ」を選びます。

## 内容

- `pet.json` — Codex用のペット定義
- `spritesheet.webp` — 8列 × 11行のアニメーションスプライトシート
